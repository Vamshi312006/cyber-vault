

## Goal

Become a cyber security engineer whose work demonstrates engineering maturity beyond the average student within the next 18 months.

---

## Philosophy

- Build evidence, not just knowledge.
- Optimize for depth over quantity.
- One effort should generate multiple hiring signals.
- Focus on long-term engineering instead of short-term resume padding.

---

## Target Identity

- Detection Engineering
- Blue Team Automation
- Security Engineering

---

## Long-Term Outcome

A recruiter should think:

> "This student already builds software like a junior security engineer."

---

## Success Criteria

- Strong engineering foundations
- Flagship project
- Open-source contributions
- Detection portfolio
- Professional GitHub
- Technical communication
