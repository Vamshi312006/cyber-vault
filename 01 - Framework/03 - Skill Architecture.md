# Purpose

Skills are the foundation of this career system.

Projects, Cyber Act, HTB, documentation, certifications, and interviews exist to develop, reinforce, and demonstrate these skills.

Every activity performed throughout this roadmap must map to one or more skills defined here.

# Success Criteria

A skill is considered complete when all of the following are true.

✓ Understand the concept

✓ Can explain it

✓ Can implement it

✓ Can debug it

✓ Can defend design decisions

✓ Can demonstrate it through a project

✓ Can discuss it confidently during interviews

Only then is the skill considered part of permanent knowledge.
